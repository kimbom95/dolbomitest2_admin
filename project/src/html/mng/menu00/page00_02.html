Color
버튼은 크게 사용 용도와 위치로 구분합니다.
먼저는 색깔 구분에 대해 설명합니다.
검정 (dkBlue) : 등록, 목록가기 등으로 해당 페이지가 전환되는 버튼입니다.
남색 (purple) : 페이지 내에서 요소가 추가 되거나, 그리드 및 테이블 등에서 바로 요소가 추가되는 경우 사용합니다.
빨강 (red) : 삭제하거나 취소등 부정적인 내용으로 사용자가 인지해하고 주의를 해야 할 때 사용합니다.
라인 (gLine) : 페이지 자체에 영향을 미치지 않는 요소에 사용합니다.
사용 위치 (그리드, 테이블 위)
- 그리드나 테이블 위에 사용
- 우측에 사용
- 버튼의 Class는 "srhBtn"를 사용합니다.
- 건수보기 등 조합으로 사용하므로 class "opList+item"를 사용합니다.
사용 위치 (그리드, 테이블 아래)
- 그리드나 테이블 아래 사용
- 좌측, 우측, 좌우측에 사용
- 버튼의 Class는 "srhBtn"를 사용합니다.
- 다른 조합으로 사용하지 않으며 "comSideOpt"안에 바로 버튼을 사용합니다
CRUD 사용 위치 (그리드, 테이블 아래)
- 기본은 페이지 최하단에 사용
- 좌측, 우측, 좌우측에 사용 가능
- 버튼의 Class는 "crudBtn"를 사용합니다.
- 다른 조합으로 사용하지 않으며 "comSideOpt"안에 바로 버튼을 사용합니다
