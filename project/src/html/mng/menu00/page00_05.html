Grid : 일반, 버튼
그리드 내부에 무엇이 들어갈 수 있는지를 정의합니다. 버튼은 그리드용으로 따로 되어 있습니다.
*그리드 th, td에 사용하는 정렬 클래스
left : 좌측정렬
center : 가운데정렬
right : 우측정렬
Grid : input, readonly
그리드 내에서 수정 및 등록이 이루어 질 경우 내부 인풋 하나로 읽기, 쓰기가 표현될 수 있습니다.
읽기 : <input type="text" class="iptText" value="사용자명" readonly="">
쓰기 : <input type="text" class="iptText" value="사용자명">
