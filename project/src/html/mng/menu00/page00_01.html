input, form
기본 폼이며 검색, 그리드, 테이블에 동일한 폼을 사용합니다
해당 input,select는 가능한 class "frmGroup" 내부에서 사용합니다.
input의 text와 password는 class "iptText"를 사용합니다.
select는 class "iptSlt"를 사용합니다.
addClass로 크기를 조절 할 수 있으며 이를 이용해서 input,select 등을 조합 할 수도 있습니다.
date
date input은 toast ui를 사용하며 아래와 같이 사용합니다.
input 크기는 class"tui-datepicker-input"에 추가합니다.
크기 class name은 다른 input과 동일합니다.
기본 실행은 아래 script처럼 사용합니다
다른 datepicker을 사용할경우 말씀주시면 바꿔 적용하겠습니다.
textBox
폼과 조합 할 때 사용하며 class는 "textBox"를 사용합니다.
addClass "deco"를 사용하여 "~" 혹은 "-" 같이 짧은 텍스트를 최소한의 길이로 사용할 수 있습니다.(15px)
checkbox, radio
checkbox는 "iptChk", radio는 "iptRdo"를 사용합니다.
체크박스나 라디오의 너비가 같은경우는 ul "frmChoList", li "frmChoBox"조합을 사용합니다.
이것 역시 다른 input,select와 조합이 가능합니다.
class "frmChoBox" 내부에 input을 넣을 수 있지만 이경우 frmTini~frmLg까지만 가능합니다.
수가 많아 여러줄이 될 가능성이 있는 경우 "frmGroup" class에 "multiLine"을 추가합니다.
input, form 조합
input, select등을 조합할 때는 하나의 "frmGroup" 안에 조합할 폼을 넣습니다.
정해진크기 + 100%짜리가 들어가면 100%짜리가 자동으로 화면 너비에 맞추어집니다.
100% + 100% 조합이 들어가면 자동으로 크기가 나누어집니다.
텍스트 박스(textBox)를 이용하여 조합 할 수도 있습니다.
